### 🌐 **Deployment Options**

#### **Netlify (Recommended)**:
1. Connect GitHub repository to Netlify
2. Auto-deploy configured with `netlify.toml`
3. Build command: `npm run build:netlify`
4. Publish directory: `out`

#### **Manual Deploy**:
```bash
npm install
npm run build:netlify
# Upload 'out' folder to any static host
```
